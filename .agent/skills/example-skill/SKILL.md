# Example Skill Template

This is a template skill that demonstrates the structure and format of an Antigravity skill.

## Purpose

Use this template as a starting point for creating your own custom skills tailored to your specific project needs.

## Instructions

When this skill is activated, the agent should:

1. **Understand the Context**: Read and analyze the specific requirements
2. **Follow Best Practices**: Apply industry-standard approaches
3. **Provide Clear Output**: Deliver well-structured and documented results

## Usage Example

To use this skill, simply reference it in your conversation:
- "Use the example skill to help me with..."
- "Following the example skill template..."

## Customization

To create your own skill based on this template:

1. Copy this folder and rename it to your skill name
2. Update the YAML frontmatter with your skill's name and description
3. Replace this content with your specific instructions
4. Add any helper scripts to a `scripts/` folder if needed
5. Include example files in an `examples/` folder if helpful

## Best Practices for Skills

- **Be Specific**: Clearly define what the skill does and when to use it
- **Be Modular**: Keep skills focused on a single domain or capability
- **Be Comprehensive**: Include all necessary context and instructions
- **Use Examples**: Provide concrete examples when helpful
- **Reference Tools**: Specify which tools the agent should use
